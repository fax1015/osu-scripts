guest difficulty BBCode list

outputs all guest difficulties made by a user in one text block. works best for your own guest diffs, as the api may not show all guest difficulties

modes

syntax

write normal BBCode, but with {tokens} for values that change.

Box / notice

  • {total} total map count for all years
  • {username} profile username
  • {user_id} profile numeric id

Year line

  • {year} e.g. 2026
  • {count} total map count for specific year

Beatmap line (after # osu-guest-line)

  • {url} beatmap link
  • {artist} artist name
  • {title} romanised song title
  • {version} difficulty name
  • {status_color} hex colour for rank tag
  • {status_label} rank tag status label

Example beatmap line

[url={url}]{artist} - {title} ({version})[/url][size=85][color={status_color}]({status_label})[/color][/size]

BBCode template

template for the output. you can change the template to your liking, the tokens will be replaced with the actual values.
